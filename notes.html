5 HTML tags learned
  1. ul (unordered list) - Tag which will show an unodered list.
  2. ol (ordered list) - Tag which will show an ordered and numberic list.
  3. dl (definition list) - Tag used to make a list compilation.
  4. dt (description term) - Tag which is the a term in the definition list (dl). Is the name of a new list.
  5. dd (description definition) - Tag which shows the details or a more complete definition of the description term (dt) in a description list (dl).
